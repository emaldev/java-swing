# Java Swing Practice

This repository contains my Java Swing practice projects.

The goal of this project is to learn how to build graphical user interfaces (GUI) using Java Swing and prepare for larger projects such as Tic Tac Toe and Checkers.

## Topics Covered

- JFrame
- JPanel
- JButton
- JLabel
- ActionListener
- Layout Management
- Colors
- Fonts
- Borders
- Images (ImageIcon)
- Arrays of Components
- Basic Game Logic

## Practice Projects

- Panels and Layouts
- Buttons and Events
- Tic Tac Toe
- More Swing exercises

## Technologies

- Java
- Java Swing
- VS Code

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/emaldev/java-swing.git
```

2. Open the project in your Java IDE or VS Code.

3. Compile and run any Java file with a `main()` method.

## Learning Goal

This repository is part of my Java learning journey. I use it to practice Java Swing concepts before building more advanced desktop applications and games.
